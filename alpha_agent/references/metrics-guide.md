# RD-Agent 指标解读指南

## 因子有效性指标

### IC（Information Coefficient）— 信息系数

因子值与下期收益率的截面相关系数，衡量因子对未来收益的预测能力。

| IC 范围 | 评级 | 含义 |
|---------|------|------|
| > 0.05 | 优秀 | 强预测能力，可作为核心因子 |
| 0.03 ~ 0.05 | 良好 | 有一定预测能力，可纳入因子库 |
| 0.01 ~ 0.03 | 一般 | 预测能力较弱，建议与其他因子组合使用 |
| < 0.01 | 较差 | 几乎无预测能力，不建议单独使用 |

### ICIR（IC Information Ratio）— IC 信息比率

IC 均值除以 IC 标准差，衡量因子预测能力的稳定性。

| ICIR 范围 | 评级 | 含义 |
|-----------|------|------|
| > 0.5 | 优秀 | 预测能力稳定可靠 |
| 0.3 ~ 0.5 | 良好 | 较为稳定 |
| 0.1 ~ 0.3 | 一般 | 稳定性不足，信号时有时无 |
| < 0.1 | 较差 | 极不稳定，不可依赖 |

### Rank IC — 秩相关信息系数

因子排名与收益排名的 Spearman 秩相关系数。相比 IC，对异常值更鲁棒，反映因子的非线性排序能力。

评级标准同 IC。

### Rank ICIR — 秩 IC 信息比率

Rank IC 的均值除以标准差。评级标准同 ICIR。

## 收益与风险指标

### 年化超额收益率（Annualized Excess Return）

因子组合相对基准的年化超额收益。

| 指标 | 说明 |
|------|------|
| `without_cost` | 不含交易成本，反映因子纯 alpha |
| `with_cost` | 含交易成本，反映因子实际可获取的 alpha |

**评级**：

| 年化超额 | 评级 |
|----------|------|
| > 15% | 优秀 |
| 8% ~ 15% | 良好 |
| 3% ~ 8% | 一般 |
| < 3% | 较差 |

cost 与 without_cost 差距过大（> 50%）说明因子换手率高，实操困难。

### 信息比率（Information Ratio）

年化超额收益除以年化跟踪误差，衡量每单位风险获取的超额收益。

| IR 范围 | 评级 |
|---------|------|
| > 1.5 | 优秀 |
| 0.8 ~ 1.5 | 良好 |
| 0.3 ~ 0.8 | 一般 |
| < 0.3 | 较差 |

### 最大回撤（Max Drawdown）

从历史最高点到最低点的最大跌幅，衡量极端风险。

| 最大回撤 | 评级 |
|----------|------|
| > -5% | 优秀（回撤小） |
| -5% ~ -10% | 良好 |
| -10% ~ -20% | 一般 |
| < -20% | 较差（回撤大） |

### 日收益标准差（Std）

日度超额收益的标准差，衡量波动性。越小越稳定。

## 模型训练指标

### L2 Loss（训练/验证）

| 指标 | 说明 |
|------|------|
| l2.train | 训练集 L2 损失 |
| l2.valid | 验证集 L2 损失 |

**解读规则**：
- `l2.valid` 接近 `l2.train` → 模型泛化良好
- `l2.valid` 远大于 `l2.train` → 过拟合，模型在训练集过度适配
- 两者都很大 → 欠拟合，模型容量不足或特征不足

## RD-Agent 特有指标

### 假设验证率

每轮 loop 中，假设被代码成功实现且回测有效的比例。

| 验证率 | 评级 |
|--------|------|
| > 80% | 优秀 — AI 假设质量高 |
| 50% ~ 80% | 良好 |
| < 50% | 一般 — 可能需要增加 loop_n |

### 因子演化效率

每增加一轮 loop，IC 的边际提升。

- 前 3 轮提升明显：说明演化有效，可继续增加轮数
- 3 轮后趋于平缓：当前搜索空间已充分探索，可停止
- IC 下降：可能出现过拟合，建议回退到最佳轮次

## 综合解读规则

### 单因子评估流程

1. **先看 IC / Rank IC**：判断是否有预测能力（IC > 0.01 才值得继续）
2. **再看 ICIR / Rank ICIR**：判断预测是否稳定（ICIR > 0.1 基本可用）
3. **看年化超额**：判断实际收益水平
4. **对比 with_cost vs without_cost**：判断交易成本影响
5. **看最大回撤**：判断极端风险是否可控

### 综合评级矩阵

| IC > 0.03 | ICIR > 0.3 | 年化 > 8% | 回撤 > -10% | 综合评级 |
|:---------:|:----------:|:---------:|:-----------:|:--------:|
| Y | Y | Y | Y | **优秀** — 可直接用于实盘 |
| Y | Y | Y | N | **良好** — 需加风控 |
| Y | Y | N | — | **一般** — 收益不足，组合使用 |
| Y | N | — | — | **较弱** — 信号不稳，谨慎使用 |
| N | — | — | — | **无效** — 弃用 |

### 多因子组合建议

- IC 相关性低（< 0.3）的因子组合效果更好
- 动量因子 + 价值因子通常低相关，适合组合
- 同类因子（如多个动量因子）相关性高，选最优即可
