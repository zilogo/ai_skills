# Stock Agent — 你的 AI 美股投资助手

## 一句话介绍

**Stock Agent 是一个完整的 AI 投研团队**——它背后有四大引擎协同工作：18 位投资大师帮你快速会诊、深度研究引擎帮你写出机构级研报、量化回测引擎帮你验证策略、模拟盘帮你零风险实战。你只需要用日常对话告诉它你想了解哪只股票，剩下的交给它。

## 引擎一：AI 投资大使团 (AI Hedge Fund) — 18 位投资大师同时会诊

### 它是什么？

想象你把一只股票同时交给巴菲特、芒格、Michael Burry、彼得·林奇……让他们每个人独立给出买入还是卖出的意见，30 秒后汇总结果。这就是投资大使团引擎。

### 它怎么工作？

18 位风格迥异的投资专家**同时、独立**地分析同一只股票，每位都给出：
- **方向判断**：看多（bullish）、看空（bearish）、中性（neutral）
- **置信度**：0-100%，表示他对自己判断的确信程度
- **推理过程**：用一两句话解释为什么这么判断

最后系统统计所有人的意见，形成**共识投票**——当 16/18 位专家都说"卖出"时，你就知道该认真对待了。

### 18 位专家分别是谁？

**12 位投资大师**——模拟真实世界中最伟大投资者的思维方式：

| 大师 | 投资风格 | 他会告诉你什么 |
|------|---------|---------------|
| **巴菲特** (Warren Buffett) | 价值投资，长期持有 | 这家公司有没有"护城河"？ROE 够不够高？现在的价格相对内在价值贵不贵？ |
| **芒格** (Charlie Munger) | 多元思维，理性评估 | 这门生意简不简单？可不可预测？有没有安全边际？ |
| **Michael Burry** | 逆向投资，做空泡沫 | 自由现金流撑不撑得起这个市值？是不是又一个泡沫？ |
| **格雷厄姆** (Ben Graham) | 价值投资鼻祖 | 股价和"格雷厄姆数字"差多少？安全边际够不够？ |
| **彼得·林奇** (Peter Lynch) | 找十倍股 | PEG 比率合不合理？盈利增长配不配得上估值？ |
| **凯西·伍德** (Cathie Wood) | 颠覆式创新 | 这家公司的创新能力得分多少？研发投入够不够？ |
| **比尔·阿克曼** (Bill Ackman) | 激进改革 | 公司内在价值和市值差距大不大？管理层效率怎么样？ |
| **德鲁肯米勒** (Stanley Druckenmiller) | 宏观趋势 | 从宏观经济周期看，现在是买入还是卖出的时机？ |
| **菲利普·费雪** (Phil Fisher) | 成长股长持 | 毛利率趋势怎么样？运营效率在提升还是下降？ |
| **帕布莱** (Mohnish Pabrai) | 不对称押注 | 赢了能赚多少、输了会亏多少？赔率划不划算？ |
| **达摩达兰** (Aswath Damodaran) | 估值权威 | 用 DCF 模型算出来的公允价值是多少？市场是在为完美定价吗？ |
| **金君瓦拉** (Rakesh Jhunjhunwala) | 新兴市场 | 收入增长趋势怎么样？内在价值被低估了多少？ |

**6 位专业分析师**——从不同维度提供量化数据支撑：

| 分析师 | 专攻方向 | 核心产出 |
|--------|---------|---------|
| **技术分析师** | 图表和指标 | RSI 超买超卖、均线趋势、布林带位置、动量强弱 |
| **基本面分析师** | 财报数据 | ROE、净利润率、收入增长率、市盈率 |
| **成长性分析师** | 增长潜力 | 营收增速、EPS 趋势、内部人买卖动向 |
| **估值分析师** | 定价合理性 | DCF 折现估值、所有者盈利估值、EV/EBITDA 相对估值 |
| **情绪分析师** | 市场情绪 | 公司内部人（高管/董事）最近在买还是在卖？ |
| **新闻分析师** | 舆情风向 | 最近的财经新闻是正面还是负面？ |

### 实际效果是什么样的？

以特斯拉（TSLA）为例，18 位专家的会诊结论：

> **巴菲特**：看空（88%）—— "ROE 仅 4.8%，盈利负增长，护城河有限。以内在价值 70 倍的价格交易，严重高估。"
>
> **Michael Burry**：看空（72%）—— "自由现金流收益率 0.4%，荒谬的估值。市值 1.5 万亿美元，自由现金流却微乎其微。"
>
> **格雷厄姆**：看空（90%）—— "股价 $449 vs 格雷厄姆数字 $26，溢价 17 倍。完全不满足安全边际。"
>
> **彼得·林奇**：看空（72%）—— "市盈率 394 倍且盈利负增长——从这里开始不可能成为十倍股。"
>
> **估值分析师**：看空（100%）—— "DCF 估值 430 亿 vs 市值 1.5 万亿，高估 97.1%。"
>
> **技术分析师**：看多（21%）—— 唯一看多的声音，但置信度很低。
>
> 最终共识：**18 位中 16 位看空**，综合决策：**卖出**，置信度 90%。

30 秒就能得到这样的结论，再也不用一个人对着 K 线图发呆了。

### 自动化回测：让大师们帮你管理每一天

投资大使团引擎还有一个独特能力——**自动化回测**。

它会模拟这样的场景：假如在过去的每个交易日，都让 18 位大师分析一次，然后根据他们的综合意见决定买卖，结果会怎样？

> 以 TSLA 为例：11 个交易日中，大师们始终保持看空共识，全程没有建仓。
> 最终净值 $100,000——没亏也没赚。

**零交易也是一种智慧**——它证明了在不确定的市场中，"不动"有时候就是最好的策略。而这正是 16 位大师的一致判断。

## 引擎二：AI 深度投研分析团队 (TradingAgents) — 机构级深度研究

### 它是什么？

如果说投资大使团是"快速会诊"，那深度投研分析团队就是"全面体检"。它会花 3-10 分钟，像一个专业投研团队一样，从四个维度对一只股票做彻底的深度研究，最终产出一份**机构级别的研究报告**。

### 它怎么工作？

深度投研分析团队会依次完成以下分析流程：

```
技术分析 → 社交情绪 → 新闻分析 → 基本面分析 → 多空辩论 → 风险评估 → 最终决策
```

每个阶段都产出一份独立的详细报告，最终合并为一份完整的研究报告。

### 四维深度分析

| 维度 | 分析什么 | 报告篇幅 | 示例发现 |
|------|---------|---------|---------|
| **技术面** | 均线排列、RSI、布林带、支撑/阻力位、MACD、成交量 | ~11,000 字 | "10 日均线跌破 50 日均线，短期下行趋势确认。布林带压缩 42%，大幅突破在即。支撑位 253-254，阻力位 276-277。" |
| **情绪面** | 社交媒体讨论热度、投资者情绪倾向、市场心理 | ~16,000 字 | "市场对 AI 战略延迟感到失望，但品牌忠诚度仍然强劲。散户情绪从狂热转向谨慎。" |
| **新闻面** | 近期重大新闻、政策变化、行业动态、地缘政治 | ~15,000 字 | "宏观滞胀环境（油价 $119 + 失业率 4.4%）加剧估值压力。防御性定价策略暗示需求疲软。" |
| **基本面** | 财报分析、现金流、盈利能力、竞争格局、行业地位 | ~17,000 字 | "毛利率 48.2% 依然出色，但 33 倍市盈率在利润率压缩环境下不可持续。PEG 2.34 偏高。" |

四份报告加起来超过 **60,000 字**——这是你在任何免费平台上都看不到的深度。

### 最精彩的环节：多空辩论

四维分析完成后，深度投研分析团队会安排一场 **多空辩论**——一位"多头分析师"和一位"空头分析师"像真正的基金经理一样激烈交锋，最后由"裁判"做出裁决。

> **多头**："看看实际数据吧——15.7% 的季度收入增长、480 亿美元的回购力度，你说这是衰退？你在混淆短期噪音和结构性趋势。"
>
> **空头**："你在混淆好公司和好股票。PEG 2.34 意味着你在为 6.4% 的年增长付出成长股的价格。高估值陷阱就是这样形成的。"
>
> **裁判**："多头押注品牌护城河和回购力度，空头警告估值陷阱和技术面疲软。综合来看，建议减仓 30%，而非全部清仓。"

这种观点碰撞能帮你看到单一视角容易忽略的盲区。

### 最终交易决策

辩论结束后，系统会给出一个**可执行的交易计划**——不只是笼统的"买"或"卖"，而是具体到：

> **建议：卖出（减仓 30%）**
>
> 1. 立即减仓 30%，锁定利润
> 2. 止损位调整到 $262（50 日均线）
> 3. 如果跌到 $240-245 区间，可以考虑回补
> 4. 如果突破 $278（布林带上轨），在回调时重新入场

这种细致到价位的操作建议，是深度投研分析团队相比投资大使团的独特价值。

### 两个分析引擎的互补关系

| 对比项 | 投资大使团 | 深度投研分析团队 |
|--------|--------------|---------------|
| **速度** | ~30 秒 | 3-10 分钟 |
| **产出形式** | 量化信号（方向 + 置信度） | 深度文本报告（60,000+ 字） |
| **核心价值** | "多少位大师看多/看空" | "在哪个价位买卖、为什么" |
| **适合场景** | 快速判断一只股票值不值得深入研究 | 已经有兴趣，需要详细分析和操作建议 |

**两者配合使用效果最佳**：

- 投资大使团告诉你："巴菲特中性 55%、Burry 看空 85%、16/18 位看空"
- 深度投研分析团队告诉你："50 日均线 $263.63、RSI 45.49、布林带下轨 $253.70、建议减仓 30%"
- **交叉验证**：两者方向一致 → 强信号；方向冲突 → 需谨慎

你不需要手动选择用哪个引擎——Stock Agent 会根据你的需求自动调度，也支持同时启动两个引擎并行分析。

## 引擎三：回测引擎 — 用历史数据验证你的想法

### 它是什么？

有了分析结论，下一步自然是——**怎么操作？** 回测引擎的作用就是：把你的交易想法变成可量化的策略，然后用真实的历史数据跑一遍，看看到底能不能赚钱。

### 它怎么工作？

两种回测模式，满足不同需求：

**模式一：自动策略回测**

投资大使团的 18 位大师在回测期间的每个交易日都重新分析一次，根据当天的共识意见自动决定买卖。你什么都不用做，它会自动模拟"如果大师们每天帮你操盘，结果会怎样"。

**模式二：自定义策略回测**

Stock Agent 综合两个分析引擎的结论，为你自动设计一套交易策略，然后提交给回测引擎验证。

**整个过程你只需要说**：
> "我想设计一个策略来交易特斯拉"

**它会帮你完成**：

1. **分析股票** → 找到关键价位和趋势方向
2. **设计策略** → 比如"当价格跌到布林带下轨且 RSI 低于 35 时买入，回到中轨时卖出，止损 2.5%"
3. **历史回测** → 用过去的真实行情数据跑一遍

### 回测结果一目了然

| 指标 | 含义 | 示例 |
|------|------|------|
| 总收益率 | 策略赚了多少 | +1.08% |
| Sharpe 比率 | 每承担一单位风险赚了多少（越高越好） | 1.76 |
| 最大回撤 | 过程中最多亏过多少 | -0.76% |
| 交易轮次 | 一共买卖了几次 | 6 轮 |
| 胜率 | 盈利次数占比 | 67%（4 赢 2 亏） |

还能看到每一笔交易的详细记录：

> 11/14 入场买入 → 11/20 均线死叉止损退出（小亏）
> 11/21 再次入场 → 11/24 布林带上轨附近获利了结
> 11/25 新信号触发 → 12/03 获利退出
> ……6 轮交易，4 赢 2 亏，最终 +1.08%

**不满意？** 你可以调整策略参数，再跑一次。反复验证，直到找到适合你的策略。

### 策略参数从哪来？

策略不是凭空编造的——每个参数都有据可循，来自两个分析引擎的结论：

| 策略参数 | 来源 |
|----------|------|
| 入场价位：布林带下轨 $253 | 深度投研分析团队技术分析报告 |
| RSI 阈值：35 | 深度投研分析团队："下跌趋势中 35-40 常作为支撑" |
| 退出条件：布林带中轨 | 标准均值回归策略 |
| 止损幅度：2.5% | 苹果波动率低于特斯拉，可以更紧凑 |
| 仓位上限：40% | AI 仍建议卖出，不宜激进 |

**分析 → 策略 → 验证**，环环相扣，逻辑清晰。

## 引擎四：模拟盘交易 — 零风险实战练手

### 它是什么？

分析完了，策略验证了，接下来想真正操作一下？模拟盘引擎连接了一个拥有 **10 万美元虚拟资金**的真实模拟交易账户，让你在零风险的环境下实战练手。

### 日常对话就能交易

| 你说 | 它做 |
|------|------|
| "买入 1 股苹果" | 以市价买入 1 股 AAPL |
| "在 250 块限价买苹果" | 设置限价单，等价格到 250 再买 |
| "苹果现在多少钱" | 查询实时报价 |
| "看看我的持仓" | 展示当前持有的股票和盈亏 |
| "平掉苹果" | 卖出所有 AAPL 持仓 |
| "看看我的账户" | 展示总资产、购买力、收益率 |

### 安全机制

这不是一个"说买就买"的危险工具。它有多重安全保障：

- **二次确认**：每笔交易下单前，都会先展示订单详情，等你说"确认"才真正执行
- **方向冲突警告**：如果 AI 分析说"卖出"但你要买入，会明确提示"AI 建议与你的操作方向相反，请确认是否继续"
- **智能仓位建议**：根据 AI 的置信度自动推荐投入比例——置信度越高，建议比例越大；置信度低于 40%，会建议你不要交易

### 分析驱动的交易

模拟盘不只是一个下单工具——它和分析引擎深度联动：

> 你说"帮我看看苹果，如果值得买就买入"
>
> Stock Agent 会先调用分析引擎 → 得到结论 → 如果建议买入，计算合理仓位 → 展示分析理由和订单详情 → 等你确认 → 执行

从分析到下单，一气呵成。

## 六种使用方式

四大引擎通过不同的组合，形成六种使用方式，覆盖投资决策的每个场景：

### 1. 全自动分析 + 回测

**适合**：快速了解一只股票值不值得买

**你说**："帮我看看苹果"

**它做**：投资大使团大师会诊 → 自动化回测 → 生成综合报告

30 秒出分析结论，几分钟完成回测验证。最适合日常快速筛股。

### 2. 双引擎深度分析 + 自定义策略 + 回测

**适合**：想深入研究一只股票，并设计自己的交易策略

**你说**："我想设计一个均线策略来交易苹果"

**它做**：投资大使团 + 深度投研分析团队双引擎并行分析 → 综合结论生成策略 → 回测引擎验证 → 综合报告

两个引擎交叉验证，策略参数有理有据。这是最全面的分析模式。

### 3. 纯分析（不回测）

**适合**：只想了解一只股票的当前状态

**你说**："分析一下英伟达最近的走势"

**它做**：根据你的需求选择引擎——

| 你的表述 | 使用引擎 | 耗时 |
|----------|---------|------|
| "分析一下 AAPL" | 双引擎（最全面） | 3-10 分钟 |
| "详细分析 AAPL 的技术面" | 深度投研分析团队（深度报告） | 3-10 分钟 |
| "大师们怎么看 AAPL" | 投资大使团（快速会诊） | ~30 秒 |

### 4. 纯回测（已有策略）

**适合**：已经有交易想法，想用历史数据验证

**你说**："用我的策略回测一下特斯拉 12 月的表现"

**它做**：直接提交策略到回测引擎 → 绩效报告

### 5. 多股对比

**适合**：几只股票拿不定主意，想横向对比

**你说**："对比一下 TSLA、AAPL、NVDA、AMD 哪个更值得买"

**它做**：对每只股票分别分析 → 汇总对比 → 排名 + 组合配置建议

实际对比结果示例：

| 排名 | 股票 | AI 决策 | 核心理由 |
|------|------|---------|---------|
| 1 | **AMD** | 持有 | 唯一非卖出信号，PEG 0.60 暗示低估，均线金叉完好 |
| 2 | AAPL | 卖出 | 基本面最稳健（毛利率 48.2%），但估值偏高 |

## 它的独特之处

**1. 不是一个引擎，而是四个引擎协同**

市面上有做分析的工具、有做回测的平台、有做交易的软件——但它们是割裂的。Stock Agent 让四个引擎在同一个对话中无缝协作：分析的结论直接变成策略的参数，策略的结果直接对接交易的执行。

**2. 不是一个人的意见，而是 18 位专家的共识**

单一分析师可能有偏见。18 位风格迥异的投资大师独立分析后统计共识——就像一场专家投票。当 16/18 位专家都说"卖出"时，这不是巧合。

**3. 不只告诉你"买"或"卖"，还告诉你"在哪个价位、为什么"**

投资大使团给方向，深度投研分析团队给价位和理由。两者交叉验证：方向一致是强信号，方向冲突要谨慎。

**4. 先验证，再行动**

每一个交易建议都可以用历史数据回测验证。不是拍脑袋告诉你"买"，而是先证明"过去这样做能赚钱"，然后你再决定要不要执行。

**5. 对话即操作**

没有复杂的界面、没有需要学习的菜单。你怎么跟朋友聊股票，就怎么跟它说话。

**6. 完整的投资决策闭环**

```
了解 → 分析 → 设计策略 → 验证 → 交易 → 复盘
         ↑                              │
         └──────── 反复迭代 ─────────────┘
```

一个助手，覆盖投资决策的每一个环节。
